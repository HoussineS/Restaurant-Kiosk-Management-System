# Restaurant-Kiosk-Management-System

A Flutter Desktop restaurant kiosk management system built with Riverpod, SQLite, Material 3, and clean architecture.

## Current Status

Sprint 1 foundation is in progress:

- Category management
- Product management
- Local SQLite storage
- Local product image storage
- Material navigation with `MaterialPageRoute`
- Clean `domain`, `data`, and `presentation` feature structure
